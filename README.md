## Maven
Projektin compilaaminen:
```
mvn compile exec:java -Dexec.mainClass=ui.Main

```
Testit toimivat normaalisti
```
mvn test
```
ja jacoco:report:
```
mvn jacoco:report
```
sekä .jar:n generointi:
```
mvn package
```
Checkstyle:
```
mvn jxr:jxr checkstyle:checkstyle
```
JavaDoc:
```
mvn javadoc:javadoc
```
